# SwipeLingo – Product Concept

## 1. Product Concept

### 1.1 App Concept
SwipeLingo is a mobile app for expanding and reinforcing English vocabulary using flashcards. Similar to paper flashcards, the app features a word in English on one side and a native translation on the other. The core mechanic is Tinder-style swiping.

### 1.2 User Pain Point
Come across an unfamiliar word → write it down → get lost. SwipeLingo solves this problem: words are quickly added, stored, and regularly reviewed using the SRS algorithm, right in your pocket.
