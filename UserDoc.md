# SwipeLingo — Поведение карточек и алгоритм повторений

Этот документ описывает логику работы карточек для пользователя.
Используется как основа для onboarding, help-экранов и App Store описания.

---

## 1. Какие карточки вы видите при изучении

### Pile — ваша учебная стопка

Pile — это набор карточек из одного или нескольких сетов, которые вы хотите изучать вместе. В любой момент активен один Pile — именно его карточки участвуют в сессии. Переключить активный Pile можно в разделе Library.

Если ни один Pile не выбран — в сессию попадают все активные карточки из всех сетов.

### Режим All

Показывает все карточки из активного Pile со статусом Active — независимо от того, пришло ли время их повторять. Используйте этот режим чтобы пройтись по всему материалу или познакомиться с новыми карточками.

### Режим Due

Показывает только те карточки из активного Pile, время повторения которых уже наступило. Это ваша ежедневная работа: новые карточки (ещё не изученные) и карточки, которые алгоритм назначил на сегодня.

*Если до вашего настроенного времени начала занятий ещё не дошло — приложение автоматически переключается в режим All.*

### Строка прогресса

|  | Режим All | Режим Due |
|---|---|---|
| Центр | текущая / всего в сессии | текущая / всего due сегодня |
| Слева | осталось active карточек | осталось due карточек |
| Справа | выучено в этой сессии | выучено в этой сессии |

## 2. Как работают кнопки оценки (SRS / SM-2)

После того как вы перевернули карточку и увидели перевод — оцените насколько хорошо вы её знали:

### [Не знал]
Вы не вспомнили слово. Карточка вернётся **завтра**. Интервал сбрасывается до 1 дня, сложность карточки растёт.

### [Сложно]
Вы вспомнили, но с трудом. Карточка вернётся **вдвое скорее** чем планировалось. Если интервал был 10 дней — придёт через 5. Сложность карточки немного растёт.

### [Легко]
Вы знали слово уверенно. Следующий интервал вычисляется по формуле:

```
новый интервал = текущий интервал × коэффициент лёгкости
```

Типичный рост интервала для новой карточки:

```
1-я оценка Easy  → вернётся через 1 день
2-я оценка Easy  → через 6 дней
3-я оценка Easy  → через 15 дней
4-я оценка Easy  → через 38 дней
...и так далее — вплоть до месяцев и лет
```

Коэффициент лёгкости и все расчёты приложение ведёт автоматически — вы этого не видите.

---

## 3. Свайп вправо — Force Learnt

Свайп вправо означает: *«Я знаю это слово и не хочу видеть его в сессиях»*. Карточка получает статус **Learnt** и исчезает из ротации немедленно — независимо от того, когда алгоритм планировал её следующий показ.

Это ваш осознанный выбор, алгоритм его не оспаривает.

Вернуть карточку в Active можно в разделе Library → ваш сет → секция Learnt → свайп влево.

## 4. Жизненный цикл карточки

```
Новая карточка
  │  dueDate = сегодня → сразу появляется в Due mode
  ▼
[Сессия] → переворот → оценка
  │
  ├─ [Не знал]  → интервал 1 день    → вернётся завтра
  ├─ [Сложно]   → интервал / 2       → вернётся вдвое скорее
  └─ [Легко]    → интервал × easeFactor → вернётся позже
        │
        └─ повторяется вечно, интервалы растут
           1д → 6д → 15д → 38д → ... → месяцы → годы

Свайп вправо (в любой момент)
  └─ статус Learnt → исчезает из сессий навсегда*

Свайп вверх (в любой момент)
  └─ статус Deleted → исчезает из всех экранов*

* восстановить можно вручную
```

---

## 5. Статусы карточки

| Статус | Что означает | Как попасть | Как вернуть |
|---|---|---|---|
| **Active** | В учебной ротации | По умолчанию у новой карточки | — |
| **Learnt** | Пользователь убрал из изучения | Свайп вправо | Library → сет → Learnt → свайп влево |
| **Deleted** | Удалена | Свайп вверх | Settings → Deleted Cards → восстановить |

## 6. Простой режим (без SRS)

В Settings → Study можно отключить алгоритм повторений (Spaced Repetition).

**SRS включён (по умолчанию):**
Карточки показываются по расписанию. После переворота появляются кнопки Не знал / Сложно / Легко. Доступны оба режима: Due и All. В настройке Pile доступна сортировка «Hardest first». В English+ по завершении сета доступны кнопки оценки Forgot / Hard / Easy.

**SRS выключен:**
Приложение работает в простом режиме. Все карточки показываются каждый раз (режим All). Кнопки оценки не отображаются — управление только свайпами: вправо — убрать в Learnt, влево — оставить в Active, вверх — удалить. Сортировка «Hardest first» в Pile скрыта (если была выбрана — автоматически сбрасывается на Random). В English+ по завершении сета доступна только кнопка Replay.
